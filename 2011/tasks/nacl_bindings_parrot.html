Task Description

NaCL Bindings for Parrot Virtual Machine

Steps To Complete This Task

  1. Create a clone of parrot-libgit2.git on github.com
  2. Use parrot-libgit2 as a skeleton. parrot-nacl will have a very similar structure to parrot-libgit2, except it is binding to the nacl C library instead of the libgit2 C library. NaCL actually has many fewer functions that libgit2, so it will be a smaller binding.
  3. The setup.winxed file will be very similar, but information needs to be modified to reference NaCL.
  4. Implement enough of the bindings such that you can call two different NaCL functions from Winxed and get their return values.
  5. Tests should be included.
  6. A new Github repository should be created where this code will live.

Benefits

  1. NaCL is a very new and exciting cryptography library which provides crypto primitives that are very useful for doing various kinds of cryptography. Having access to this library from Parrot will allow new and interesting cryptography applications to be built on Parrot.

Requirements

  1. Knowledge of Git and build systems, C/C++
  2. Ability to learn how to use Winxed (very similar to Javascript) and use a Foreign Function Interface (FFI)
