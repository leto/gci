Task Description

Write 10 tests for Jaesop stage 0

Steps To Complete This Task

  1. Create a fork of jaesop.git(Javascript on Parrot) on github.com
  2. Write 10 new tests for the Stage 0 runtime and add them to t/stage0_runtime
  3. Build Jaesop and run it's test suite (make fulltest) to verify that things still work
  4. Create a Github pull request (button on the upper right of your fork) to have your changes incorporated into the master repository

Benefits

  1. This will increase the code coverage of Jaesop Stage 0, which is the basis of Javascript on Parrot and ensure that certain very low-level features are working correctly.

Requirements

  1. Knowledge of Javascript
