Task Description

Increase the code coverage of the String PMC by at least 5%

Steps To Complete This Task

Currently, the String PMC is at 83% code coverage, which is a bit low for such an important PMC. This task requires the student to write tests to increase the code coverage of this important PMC. The tests should be added to t/pmc/string.t
  1. Create a fork of parrot.git on github.com
  2. Perform all the necessary code and/or documentation changes
  3. Build parrot and run it's test suite (make fulltest) to verify that things still work
  4. You can also submit a smoke report with "make smoke", but that is optional.
  5. Create a Github pull request (button on the upper right of your fork) to have your changes incorporated into the master repository

Benefits

  1. This will ensure there are no hidden bugs in our Strings, which are a very important subsystem of Parrot.

Requirements

  1. Knowledge of writing tests and PIR
  2. Abillity to debug.
