Task Description

Translate README to Romanian

Steps To Complete This Task

  1. Create a fork of parrot.git on github.com
  2. Copy the README to README.$language (where $language will make sense to native speakers) and translate file to Romanian
  3. Perform all the necessary code and/or documentation changes
  4. Create a Github pull request (button on the upper right of your fork) to have your changes incorporated into the master repository

Benefits

  1. Romanian speakers will be able to read our README instructions in their native language, which makes it easier for Romanian speakers to get involved with Parrot.

Requirements

  1. Native or Fluent Romanian speaker
